# 📌 Project Overview

## About the Project

This project analyzes **India's imports from Oceanic countries** to identify import trends, country-wise trade relationships, commodity distribution, and key business insights using Python.

The project includes:

- Data Cleaning
- Data Preprocessing
- Exploratory Data Analysis (EDA)
- Statistical Analysis
- Data Visualization
- Business Insights

---

# 🎯 Objectives

- Clean and preprocess import data.
- Analyze import trends across Oceanic countries.
- Identify top importing countries.
- Study commodity-wise imports.
- Discover hidden trade patterns.
- Generate actionable business insights.

---

# 📂 Dataset

| Feature | Description |
|---------|-------------|
| Year | Import Year |
| Country | Oceanic Country |
| Commodity | Imported Commodity |
| Quantity | Import Quantity |
| Value (INR) | Import Value in INR |
| Value (USD) | Import Value in USD |
